## 📝 Project Overview

An **interactive image gallery** for Photogram with the following features:

* 🖼️ **Gallery Layout:** 3x3 grid of thumbnails (150x150px each)
* 💻 **Modal Viewer:** Click a thumbnail to open full-size image with dark overlay
* 🔤 **Typography:** Roboto Bold (24px) for headings, Arial (14px) for captions
* 🎨 **Colors:** Light cream background (#faf3e0), red accent borders (#ff6b6b)
* ✨ **Interactivity:** Smooth opening/closing of modal using JavaScript
* 📱 **Responsive:** Works on desktop, tablet, and mobile
