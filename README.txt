Photogram Gallery Project
=========================

Client: Photogram (Fictional Photo-Sharing Platform)
Project Type: Interactive Image Gallery

Project Overview:
-----------------
This project is a small, interactive image gallery designed for Photogram.
- Gallery Layout: 3x3 grid of thumbnails (150x150px each)
- Modal Functionality: Clicking a thumbnail opens the full-size image in a modal with a dark overlay
- Typography: Gallery heading in Roboto Bold (24px), captions in Arial (14px)
- Colors: Light cream background (#faf3e0), dark text, red accent borders (#ff6b6b)
- Interactivity: Smooth transitions for opening and closing the modal using JavaScript

Images:
-------
- Images used are placeholders (Nature 1-9 from public sources)
- Images can be easily replaced with client-specific images by updating the `src` attributes in HTML

How to Use:
-----------
1. Open `index.html` in a modern web browser
2. Click any thumbnail to view the larger image in the modal
3. Close the modal by clicking the close button or clicking outside the modal

File Structure:
---------------
/PhotogramGallery.zip
├── index.html       -> Main HTML file
├── style.css        -> CSS for styling the gallery and modal
├── script.js        -> JavaScript for modal interactivity
├── images/          -> Folder containing gallery images
└── README.txt       -> Project documentation (this file)

Notes:
------
- Fully responsive: images and modal adapt to different screen sizes
- Tested on major browsers (Chrome, Firefox, Edge)
